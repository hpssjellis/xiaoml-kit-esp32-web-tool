# xiaoml-kit-esp32-web-tool
xiaoml-kit-esp32-web-tool



If you have the $38.90 USD Seeedsstudio [xiaoML Kit](https://www.seeedstudio.com/The-XIAOML-Kit.html) then this webpage is essential for you.

Main demo page here:  https://hpssjellis.github.io/xiaoml-kit-esp32-web-tool/public/index.html

Basically plug your Esp32S3 Sense (xiaoML kit has the OLED / IMU module) into your computer and load the above webpage and program and interact with your device.

The files webSerial## are the latest webSerial to XIAOML kit connection which is duplicated on the index.html file

The binary files can be flashed to the xiao ML kit 

The arduino IDE code is called   esp32-to-web##.txt  with the latest one having the largest ##.

Quick Commands
Quick:

Uses esptool-js v0.5.7 to flash via WebSerial directly from the browser. Serial monitor is released automatically before flashing and reconnected after. Requires HTTPS or localhost.

Option A — Sketch Only (.bin)

Flash only the application partition at 0x10000. Fast. Preserves bootloader and NVS.


Default: esp32_web_tool_test07.ino.bin — browse to override

Option B — Full / Merged Flash (.bin)

Flash a merged binary combining bootloader, partition table, and app. Use for fresh chips.


Default: esp32_web_tool_test07.ino.merged.bin → 0x0 — browse to override


⚠ MUST REFRESH WEBPAGE FIRST, then:

(replug ESP32 after flashing)
